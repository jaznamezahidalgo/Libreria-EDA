## Entorno virtual

$ conda create —name eda_env python=3.7
$ conda activate eda_env

En el entorno deben estar las lineras básicas para trabajar en Ciencia de datos

python 3.7.12
pandas 1.3.5
numpy 1.21.5
matplotlib 3.5.1
seaborn 0.11.2

El comando que me permite comprobar las instalaciones

$ conda list
